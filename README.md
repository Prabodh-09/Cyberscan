GameSecure is a prototype Android application that scans gaming applications for basic cybersecurity threats and generates security assessment reports.

## Features
- Game/application security scanning  
- Detection of common threat indicators  
- Risk level classification (Low / Medium / High)  
- Automated security report generation  
- Lightweight, non-intrusive analysis  

## Tech Stack
- Android Studio  
- Java / Kotlin  
- Static Security Analysis  
- Report Generation  

## Platform
- Android
